## Описание программы
*"Book Tracker — это GUI-приложение для управления списком прочитанных книг. Позволяет
добавлять записи о книге с возможностью фильтрации, сохранением данных в JSON."*

## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше
- Дополнительные библиотеки не требуются (используются только встроенные модули: Tkinter, JSON, Random)
